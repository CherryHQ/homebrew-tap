# CherryHQ Homebrew Tap

Homebrew formulae for [CherryHQ](https://github.com/CherryHQ) projects.

## Usage

```bash
brew tap CherryHQ/tap
```

## Formulae

| Formula | Description | Install |
|---------|-------------|---------|
| [stella](https://github.com/CherryHQ/stella) | Self-hosted AI assistant | `brew install stella` |

## Install

```bash
# Tap and install
brew tap CherryHQ/tap
brew install stella

# Or in one step
brew install CherryHQ/tap/stella
```

## Run as a service

```bash
brew services start stella
```

## Update

```bash
brew update && brew upgrade stella
```
